

-다국어 설정 (strings.xml)

res 디렉터리에 있는 strings.xml에는 string tag를 이용하여 여러 문자열을 미리 지정할 수 있음.

프로젝트에 들어가는 문자열들을 위의 strings.xml의 string 태그에 있는 내용들을 호출하는 방법으로 

작성하는 것으로 다국어 지원이 가능함.

	(eg. 	영어는 영문이 담긴 strings.xml에서 호출하고, 한국어는 한글이 담긴 strings.xml에서 호출하는 식. 
		단, 이와 같이 사용하기 위해선 서로다른 string 태그의 선택자명이 두 xml 파일 모두 같아야 함.)

-strings.xml 설정 방법

	1) 안드로이드 스튜디오 툴에서 res/values/strings.xml 파일 우클릭 후 new - values resource file 선택

	2) 새로 뜨는 New Resource File 윈도우에서 좌측 Available qualifies - Locale 탭 선택

	3) 중앙의 Language 탭에서 만들고 싶은 언어를 선택

	4) 디렉터리명은 툴이 알아서 생성, 파일명만 지정해주고 OK 클릭

	5) 이후 생성된 xml 파일을 확인 하고 열어서 안의 <string> tag에 작성하고픈 문자열 작성 

	6) 프로젝트가 작성한 언어권 xml 파일을 잘 불러오는 지 보기 위하여 가상머신을 켠 후 설정의 언어 설정을 눌러, 작성한 언어권 국가를 선택 후 앱 실행

예제 프로젝트 조건

1) page_1 / page_1_landscape / page_2 의 이미지와 같이 화면 디자인

2) page_1_landscape는 page_1의 가로화면을 의미. (가상머신으로 앱 실행하였을 때 가로화면으로 돌렸을 때 나와야 함)

3) string.xml을 이용한 다국어 설정 (default 언어 + 기타 외국어)

4) 가입을 위해 기재하는 edit text 위젯에 포커싱 하였을 때 입력도구가 올라왔을 때의 문제 해결
